# 🌤️ Weather Forecast App

This is a simple JavaScript-based weather forecasting web application that fetches and displays **current weather data** for any user-provided location using the **WeatherAPI**.

---

## 🚀 Features

- Fetches **real-time weather data**
- Displays:
  - Temperature (°C)
  - Location (city, region, country)
  - Last updated time
- Uses **async/await** for API calls
- Clean, minimal UI with **favicon integration**
