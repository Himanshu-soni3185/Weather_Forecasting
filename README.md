# WeatherNow

A modern, Glass UI inspired weather forecasting web application.

## Features

- **Current Weather**: View the current weather conditions, including temperature, "feels like" temperature, humidity, wind, pressure, and visibility.
- **Search**: Search for weather in any city using the built-in search functionality.
- **Geolocation**: Use your device's current location to get local weather data (requires HTTPS or `localhost`).
- **Forecasts**: Access detailed hourly and 7-day weather forecasts.
- **Unit Toggle**: Easily switch between Celsius (°C) and Fahrenheit (°F).
- **Beautiful Design**: A modern, responsive glassmorphism user interface.

## Tech Stack

- **HTML5**: Semantic markup.
- **CSS3**: Custom styling with a glassmorphism aesthetic.
- **JavaScript (Vanilla)**: Handles API requests, UI updates, and application logic.
- **API**: Powered by the [Open-Meteo](https://open-meteo.com/) free API (no API key required) for weather data and geocoding.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your web browser.
*(Note: For the geolocation feature to work, you may need to serve the files over `localhost` using a local web server (like VS Code Live Server), or via HTTPS if hosted online).*
